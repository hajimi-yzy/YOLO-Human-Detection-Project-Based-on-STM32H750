# V7.3 更新说明

## ESP32-S3 固件

- 增加 L610 4G 状态、频段、小区信息查询和设置指令。
- 增加按需启用的 Wi-Fi 扫描、连接和云端链路切换。
- 增加 5/8/15/20/30 FPS 与 VGA/HD/FHD 视频设置。
- 保持 SoftAP 本地 MJPEG 与 L610 云端视频并行工作。
- 修复 RNDIS 接收缓冲区分配失败后的包校验和恢复逻辑。
- 保持现有 ESJP、ESUT、ESCTL 和 STM32 UART 帧格式不变。

## 服务端

- 增加 4G、Wi-Fi 和视频参数的控制及状态接口。
- 增加基站信息解析、基站定位查询、结果缓存和 GPS 优先的位置接口。
- 支持 FHD JPEG 帧大小上限，并保留现有 MJPEG 发布链路。
- 增加对应单元测试。

## Web 前端

- 增加 4G 管理、Wi-Fi 管理和视频流设置页面。
- 地图支持基站定位开关、GPS 优先与基站结果回退。
- 操作结果写入右侧事件记录。
- 普通 GET 请求不再无条件附加 JSON Content-Type，避免不必要的 CORS 预检。
